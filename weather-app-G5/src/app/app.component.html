Instructions

Create an Angular application that will interact with the Geocoding and Weather service to populate the data needed for the current weather and forecast. All the data that you will need can be retrieve in the API services describe in the Functional Requirement below.

You will need to get API Keys for the ff. services, sign up for a FREE account:

- Open Weather Map https://openweathermap.org/

- Map Box https://www.mapbox.com/


Functional Requirement:

  1. On page load, your browser must be able to determine your location, use Geolocation.getCurrentPosition(). Using the latitude and the longitude, query the Geocoding API to get the place name to displayin the .location h2

  2. User must be able to change the location using the Search location text input. Using the input provided by the user, query the Map Box's Geocoding API to get the place name to display in the .location h2

  3. Using the latitude and the longitude, query the Open Weather Map's One Call API to fetch for the current weather and forecast of that location.

Geocoding API

GET https://api.mapbox.com/geocoding/v5/mapbox.places/[location-to-search|long,lat].json?access_token=(mapbox-api-key)&limit=1

Sample Response:

{{ mbData | json }}

One Call API

GET https://api.openweathermap.org/data/2.5/onecall?lat=(latitude)&lon=(longitude)&exclude=minutely,hourly&appid=(openweathermap-api-key)

Sample Response:

{{ owmData | json }}

Refer to the documentation for more details

https://docs.mapbox.com/api/search/geocoding/

https://openweathermap.org/api/one-call-api
